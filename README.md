# 🏠 Airbnb Global Market Performance Dashboard

An interactive Power BI dashboard analyzing **250K+ Airbnb listings and 5M+ reviews across 10 global cities**.  
The project explores pricing behavior, listing distribution, and demand trends to uncover insights about the global short-term rental market.

---

## 📊 Project Overview

This dashboard transforms large Airbnb datasets into clear market insights.

It allows users to quickly explore:

- Market differences between cities
- Factors influencing Airbnb prices
- Review activity trends & seasonality
- Which cities offer better value for travelers

---

## 🛠 Tech Stack

- **Power BI Desktop** – Dashboard design & visualization  
- **Power Query** – Data cleaning and transformation  
- **DAX** – Calculated measures and KPIs  
- **Data Modeling** – Relationships between listings and reviews tables  

---

## 📂 Data Source

**Airbnb Listings & Reviews Dataset**  
Source: Maven Analytics Data Playground  

https://mavenanalytics.io/data-playground/airbnb-listings-reviews

Dataset includes:

- 250,000+ Airbnb listings  
- 5,000,000+ guest reviews  
- Data from 10 major global cities  

### Tables Used

**Listings**
- listing_id
- host_id
- city
- neighborhood
- room_type
- price

**Reviews**
- listing_id
- reviewer_id
- review_date

---

## 🎯 Key Questions

This analysis focuses on answering the following questions:

- How do Airbnb markets differ across cities?
- Which listing attributes influence price the most?
- Are there seasonal trends in review activity?
- Which cities offer better value for travelers?

---

## 📈 Dashboard Highlights

**Market Comparison**  
Compare the number of Airbnb listings across cities to identify major Airbnb markets.

**Pricing Analysis**  
Analyze how room types and location influence listing prices.

**Demand Trends**  
Identify seasonal patterns in tourism demand using historical review data.

**Geographic Distribution**  
Visualize the concentration of Airbnb listings across different city locations.

---

## 💡 Key Insights

- Room type strongly impacts pricing, with entire homes being the most expensive.
- Listing supply varies significantly across cities, showing different market sizes.
- Review activity reveals seasonal tourism demand patterns.
- Some cities offer significantly better price-to-value options for travelers.
